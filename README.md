# 📰 Notícias Cidade

Interface web desenvolvida com foco em **estruturação semântica, organização arquitetural e boas práticas de Front-End**.

Este projeto simula um portal de notícias e foi construído com **HTML5 e CSS3**, priorizando clareza de código, separação de responsabilidades e base escalável para evolução futura com JavaScript ou frameworks como React.

---

## 🎯 Objetivo do Projeto

Demonstrar domínio em fundamentos essenciais do desenvolvimento Front-End:

* Estruturação semântica com HTML5
* Organização de layout inspirado em portais reais
* Separação clara entre estrutura e estilo
* Código limpo e de fácil manutenção
* Base preparada para evolução técnica

---

## 🧱 Estrutura do Projeto

O projeto foi organizado de forma simples e escalável:

* `index.html` → Estrutura principal da aplicação
* `css/style.css` → Camada de estilização separada
* `assets/` → Recursos visuais (imagens)

Essa organização facilita manutenção, leitura e crescimento do projeto.

---

## 🖥️ Arquitetura de Interface

A interface foi estruturada seguindo padrões utilizados em portais profissionais:

* Header com identidade visual
* Menu de navegação claro e categorizado
* Área principal de conteúdo
* Sidebar para informações complementares
* Footer estruturado

O foco foi criar uma hierarquia visual clara e boa escaneabilidade de conteúdo.
